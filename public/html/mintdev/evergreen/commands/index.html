Evergreen Commands


Evergreen provides simple command structure as well as amazing
documentation to help provide and easy exprience for the end user


Core Commands

e!help
Provides a list of Evergreens current built in commands
Syntax:
e!help | e!help [command]
Aliases:
e!help | e!h | e!command | e!commands
e!poll
Initiates a reaction poll
Syntax:
e!poll [Poll Text]
e!inventory
Shows the users inventory
Syntax:
e!inventory
e!store
Shows the server's store
Syntax:
e!store
e!buy
Purchase an item from the store
Syntax:
e!buy

Moderation Commands

e!info
Gets user info of the mentioned user
Syntax:
e!info [user]
e!purge
Purges/Deletes messages from the current channel
Syntax:
e!purge [amount]
e!mute
Mutes the mentioned user
Syntax:
e!mute [user]
e!unmute
Unmutes the mentioned user
Syntax:
e!unmute [user]

Administration Commands

e!ban
Bans mentioned user
Syntax:
e!ban [user]
e!kick
Kicks the mentioned user
Syntax:
e!kick [user]
e!logchannel
Sets the Evergreen log channel for your server
Syntax:
e!logchannel [channel]
e!prefix
Sets the Evergreen prefix for your server
Syntax:
e!prefix [prefix]
e!cmdadd
Creates a custom server command
Syntax:
e!cmdadd [command] [response]
e!cmdremove
Removes a custom server command
Syntax:
e!cmdremove [command]
e!captcha
Enables or Disables Captcha verification for your server
Syntax:
e!captcha enable/disable
