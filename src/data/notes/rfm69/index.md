# RFM69 Research for Flipper Zero

This page collects my experiments and references for RFM69 and Flipper Zero.

- Modulation: GFSK_Rb250Fd250
- Frequency: 915.0 MHz (US)
- Bit Rate: 250 kbps
- Deviation: 250 kHz
- RX Bandwidth: 500 kHz
- TX Power: 13 dBm
- Sensitivity: -120 dBm
